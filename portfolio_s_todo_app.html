Todo App avec Django

Application "Liste de choses à faire"

Cette application m'a permis de mettre en pratique les notions apprises avec mes cours sur le framework Django.
C'est une petite application qui permet d'ajouter ou de retirer des éléments dans une base de données, de barrer une tâche ou d'éliminer la barre suivant si une tâche a été réalisée ou non et de renommer une tâche. J'ai mis en place des boutons pour rendre possibles ces actions.

Consulter le code du projet via Github
Technologies utilisées : HTML, CSS, Django, Boostrap 5




  • 1
    Application facile d'utilisation

    Application simple et claire à utiliser. Les boutons indiquent la fonction qui sera éxécutée.

  • 2
    Mise en page aérée

    L'utilistaeur ne "subit" pas la page et peut évoluer dans le site tranquillement. Le renommage de la tâche se fait sur une nouvelle page.

  • 3
    Tirer avantage de Django et Bootstrap

    Fonctionnalités optimum et aspect visuel propre et clair .
