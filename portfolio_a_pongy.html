Pongy avec le module Pygame

Pongy réalisé avec Pygame

Deuxième jeu que je conçois. Ayant réalisé le même jeu avec turtle et ayant un soucis de freeze entre l'image et le son, j'ai recodé le jeu avec Pygame.
J'avais déjà un peu utilisé Pygame dans d'autres projet mais pas dans un contexte de jeu, pour utiliser la fonction mixer qui permet de jouer de la musique Plus de problèmes "Images/sons" et tout fonctionne parfaitement!!

Technologies utilisées : PYTHON, Pygame Consulter le code du projet via Github
