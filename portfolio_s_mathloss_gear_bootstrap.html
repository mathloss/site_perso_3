Site utilisant Bootstrap - Carroussel

Ce site estt une application pratique de mon apprentissage de Bootstrap.
Ce site m'a permis de mettre en place une nouvelle façon de concevoir une interface de site en apportant des concepts plus "modernes" et dynamiques.
J'ai pu utiliser les carroussels, les cards et tous les avantages que peut apporter Bootstrap.

Consulter le code du projet via Github
Technologies utilisées : HTML, CSS, BOOTSTRAP

    Site modifiable et évolutif
