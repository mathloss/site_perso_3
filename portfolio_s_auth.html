Application Authentification

Une application faite avec Django , pouvant être réutilisée pour un site web.
C'est une petite application qui permet de créer un compte en donnant des informations personnelles puis de se connecter à sce compte. La possibilité de modifier ses coordonnées et son mot de passe sont activées.
Un encart de message apparait pour avertir l'utilisateur de la démarche opérée et sa validité.

Consulter le code du projet via Github
Technologies utilisées : HTML, CSS, Django, Boostrap 5

    Application réutilisable

    Apllication qui conviendra à tous types de sites web nécessitant un compte utilisateur.
